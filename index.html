2 · Replace the example criteria

Open this file and edit the GUIDELINE object near the top of the script. Each condition has an id, name, category, and a list of criteria. Mark must-do items with critical:true. Copy the wording verbatim from the December 2025 guideline so the audit reflects the real standard.
{ id:"cap", name:"Community-Acquired Pneumonia", category:"Respiratory",
  criteria:[
    { id:"cap1", text:"Severity assessed & documented on admission", critical:true },
    { id:"cap2", text:"Empirical antibiotic matches guideline first-line choice", critical:true },
    ...
  ]
}
